# Gravity Snake

A 2D snake game based on a Roblox game with real physics.

## Made By

- [NCPlays11](https://github.com/NCPlays11)
- [Cardibro](https://github.com/Cardibro)

## Inspiration

Inspired by the concept of **Worm Tower** on Roblox.  
This project is an original implementation built from scratch, with no assets or code taken from the original game.
